- These are some project ideas you can create -

Project Prompt: All About Me!
 Difficulty: Easy

- Make a Sprite representative of you!
 - Have 3 facts about you (Favorite Artist, What you do for fun, etc.)
Tip: You can upload images from your computer into scratch to really customize your work!
 Example: All About Me

Project Prompt: Dance Battle!
Difficulty: Medium

- Create 2 contestants
 - Make them each do unique dances
 - Implement a voting system
 - Don't forget to add music!
Tip: Broadcast blocks are super helpful with this project!
 Example: America's Got Talent - Dance Off!

Photo from: Flickr
Project Prompt: Maze Game!
 Difficulty: Hard
- Have one main Sprite that you can control
 - Certain blocks, colors, or sprites leave a negative or positive effect after touching it
 - Have an end goal
 - Have a way to win and lose
 - Add cool sound effects!
 - Have a theme to your maze
 - For an extra hard challenge, add multiple levels!
Tip: Copying some of the code from the example may be useful if you're having difficulty
 Example: Ice Maze

Image from: iStock
Project Prompt: Fortune Teller!
 Difficulty: Medium
- Have one Sprite be the "Fortune Teller"
 - Have 4 options that lead to other options that end up with a fortune you write
 - Let the player interact and type to choose their fate
Tip: Look at the example for some helpful starter code!
 Example: Fortune Teller
