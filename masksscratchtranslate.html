Este video muestra cómo funcionan los bloques de control y las diferentes cosas interesantes que puedes hacer con él.

Los bloques de detección pueden ayudar a tu sprite/personaje a realizar una acción cuando tocas algo específico, como colores u otros sprites/personajes.

Los bloques de operación se usan para las entradas del jugador. Si eligen una ruta determinada o escriben algo específico, el bloque mostrará lo que se les dijo que hicieran.

Más Blocks son comandos personalizados que puedes crear, como darle a algo un nombre y una definición. Si desea que un sprite/personaje ejecute un grupo específico de comandos pero no desea volver a hacer el mismo cada vez, use More Blocks para definir un grupo de comandos para reutilizarlos fácilmente en su código.
